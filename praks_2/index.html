Võrdlused ja loogika

Tavaline võrdlus käib operaatoriga ==. Saadaval on ka üldtuntud operaatorid !=, >, <, >= ja <=

Boole'i loogikaoperaatorid - &&, || ja !

Lisaks veel and, or ning not

Toimivad samamoodi, kuid erinevused tehete järjekorras. Kindla ja enamasti ootuspärase käitumisega on esimesed

Võrdlused ja loogika

Kõiki objekte saab võrdluses kasutada, käitumine kindla reegli järgi - kõik false ja nil väärtused on false, ülejäänud true 

            
              "123" && true
              => true
              "123" and 123
              => true
              nil or false
              => false

Programmivoog

Otsustuspunktide loomine käib if lausega. Formaat on sarnane teistele keeltele, väikeste erinevustega - kasutusel on end bloki lõpetamiseks 

            
              if 2 > 3
                puts "Never going to happen"
              elsif rand>0.5
                # rand genereerib juhusliku arvu [0.0, 1.0)
                puts "Might happen"
              else
                puts "Might also happen"
              end

Programmivoog

Eksisteerib ka vastandtehe unless

Mõlemat saab kasutada rea lõpus kontrollimaks ühe lause käivitamist 

            
              arv = 10000
              puts "Päris suur number" if arv > 200

Tsüklid

Peamised vahendid while, for on Rubys olemas

            
              i = 0
              while i <= 10
                puts "i = #{i}"
                i+=1
              end

              stuff = ["first", "second", "third"]
              for item in stuff
                puts item
              end

Tsüklid

Lisaks on olemas until

            
              i = 0
              until i > 10
                puts "i = #{i}"
                i+=1
              end

Tsüklid

For enamasti ei kasutata, Ruby arendaja teeks eelneva näite pigem nii:

            
              stuff = ["first", "second", "third"]
              stuff.each do |item|
                puts item
              end

Tsüklid

Tsüklitest saab välja kasutades break, jätta praegune iteratsioon vahele kasutades next

            
              i = 0
              until i > 10
                next if i.even?
                break if i == 8
                puts "i = #{i}"
                i+=1
              end

Sümbolid

Ruby kasutab lisaks muudetavadele sõnedele ja muutmatuid sümboleid. Need defineeritakse sarnaselt sõnega, kuid kooloniga alguses 

            
              status = :ok

Kasutatakse sisemiselt olekute hoidmiseks, kus muutuv sõne võib tekitada vigu või kus pole vaja kasutajapoolset sisendit.

Massiivid ja räsitabelid

Massiivi defineerimiseks pole vaja teada suurust, defineeritakse nii: 

            
              array = []
              x = Array.new

Suurust saab dünaamiliselt muuta, pikenemisel täidetakse tühjad kohad nil väärtustega. Esimene indeks on Ruby-s 0. 

            
              x = []
              x[4] = "banana"
              x
              => [nil, nil, nil, nil, "banana"]

Massiivid ja räsitabelid

Räsitabel on järjestamata kogum võti-väärtus paaridest. Võtmetena kasutatakse tavaliselt sümboleid, räsitabel defineeritakse nii: 

            
              hash = {}
              x = Hash.new
              y = { :key => "value" }

              x[:fruit] = "banana"
              x
              => {:fruit => "banana"}

Meetodid

Meetodite defineerimine käib lausega def. Kui ei ole täpsustatud, siis defineeritakse parajasti kasutuelolevasse skoopi. 

            
              def say_hello(name)
                puts "Hello #{name}"
              end

              say_hello "Mikk"
              => "Hello Mikk"
